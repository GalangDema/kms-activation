Pendahuluan

Key Management Service (KMS) memungkinkan organisasi untuk mengaktifkan Windows dan Office tanpa harus menghubungkan komputer langsung ke server Microsoft. Aktivasi dilakukan secara otomatis melalui jaringan internal.

Persyaratan Aktivasi

  • Sistem operasi yang mendukung KMS (Windows 8 atau lebih baru, Windows Server 2012 ke atas).
  • Server KMS di jaringan lokal dengan port TCP 1688 terbuka.
  • Koneksi antara klien dan server KMS aktif.
  • Penggunaan kunci GVLK yang sesuai dengan edisi sistem operasi.

Langkah Aktivasi KMS

  1. Instal Windows atau Office edisi volume di komputer klien.
  2. Masukkan kunci GVLK yang sesuai.
  3. Pastikan komputer dapat mengakses server KMS.
  4. Jalankan perintah aktivasi: 
    slmgr /ato
  5. Periksa status aktivasi: 
    slmgr /xpr

Langkah Aktivasi KMS (Galang Version)

  1. Jalankan perintah aktivasi: 
    slmgr /upk
    slmgr /cpky
    slmgr /ipk [Serial key your windows version]
    slmgr /skms kms8.msguides.com
    slmgr /ato

Troubleshooting

  • Pastikan DNS record _VLMCS._TCP tersedia di jaringan.
  • Port TCP 1688 tidak diblokir oleh firewall.
  • Gunakan perintah nslookup -type=srv _vlmcs._tcp untuk memeriksa ketersediaan KMS server.
  • Periksa log di Event Viewer untuk detail error.
