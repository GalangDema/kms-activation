Pendahuluan

Key Management Service (KMS) memungkinkan organisasi untuk mengaktifkan Windows dan Office tanpa harus terhubung langsung ke server Microsoft. Aktivasi dilakukan secara otomatis melalui jaringan internal dengan sistem volume license.

Persyaratan Aktivasi

  • Windows 8 / Server 2012 atau lebih baru.
  • Server KMS aktif dengan port 1688 terbuka.
  • Koneksi jaringan antar perangkat lancar.
  • Gunakan kunci GVLK sesuai versi Windows.

Langkah Aktivasi KMS

  1. Instal Windows/Office edisi volume.
  2. Masukkan kunci GVLK yang sesuai.
  3. Pastikan komputer dapat mengakses server KMS.
  4. Jalankan perintah: 
    slmgr /ato
  5. Cek status aktivasi: 
    slmgr /xpr

Panduan Aktivasi KMS (Custom by Galang)

slmgr /upk
slmgr /cpky
slmgr /ipk [Serial key your windows version]
slmgr /skms kms8.msguides.com
slmgr /ato

Troubleshooting

  • Pastikan DNS record _VLMCS._TCP tersedia.
  • Pastikan port 1688 tidak diblokir firewall.
  • Gunakan nslookup -type=srv _vlmcs._tcp untuk cek server KMS.
  • Cek log Event Viewer untuk detail error.
